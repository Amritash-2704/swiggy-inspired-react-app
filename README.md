# 🍔 Swiggy-Inspired React App

A Swiggy-inspired food delivery web application built using **React.js**.
This project fetches real-time restaurant and menu data from Swiggy public APIs and displays it in a clean, user-friendly UI.

The application is developed using **StackBlitz** and focuses on learning modern React concepts like component architecture, routing, state management, and API integration.

---

## 📌 Features

* 🍽️ Browse restaurants
* 📄 View restaurant menus
* ⚡ Dynamic data fetching using API
* 🔄 Shimmer loading effect
* 🔍 Search & filter functionality
* 🧭 Client-side routing using React Router
* 📱 Responsive UI design

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Language:** JavaScript (ES6+)
* **Styling:** CSS / Tailwind CSS
* **Routing:** React Router
* **API:** Swiggy Public APIs (unofficial)
* **Development Tool:** StackBlitz

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/swiggy-inspired-react-app.git
```

### 2️⃣ Navigate to project folder

```bash
cd swiggy-inspired-react-app
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start development server

```bash
npm run dev
```

👉 App will run on:

```
http://localhost:5173
```
